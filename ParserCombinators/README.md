
# Combinatoric Parsers

Первый парсер отвечает за анализ JVM-байткода, собирает по нему важную информацию по классу, вложенная обработка ошибок, написано методом парсер-комбинаторов.

Второй парсер разбирает упрощённый вид html, в котором могут отсутствовать закрывающие теги или парные скобки, мотивировано Python библиотекой BeautifulSoup.

Общая логика вынесена в класс Parser.
